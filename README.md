# RamosVault

Premium digital accounts marketplace — Secure. Fast. Reliable.

Built with React + Vite + TypeScript + Tailwind CSS + shadcn/ui

## Features
- 12 product categories (Facebook, Instagram, TikTok, Twitter, Telegram, VPN, etc.)
- User dashboard, wallet, orders, transactions
- Admin panel with full CRUD
- PDF/Image invoice export
- Dark/light mode, search, referral system

## Stack
React, Vite, TypeScript, Tailwind CSS, shadcn/ui, Wouter, Drizzle, jsPDF, html2canvas
